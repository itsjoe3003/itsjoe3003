## 🚀 Projects

### 🔒 [Elysium Security](https://www.elysiumsecurity.us)
**Real-Time Retail Theft Detection · Natural Language Video Person Search**
`YOLOv11` `CLIP` `TorchReID` `Bi-LSTM` `TCN` `ST-GCN` `React` `GCP`

Startup building AI-powered security tools for retail environments. Two core systems:
- **Theft Detection** — real-time surveillance using YOLOv11 + gesture recognition (Bi-LSTM, TCN, ST-GCN) to identify suspicious behavior at 85% accuracy
- **Video Person Search** — natural language query system using CLIP + TorchReID to retrieve matching clips of individuals from pre-embedded video footage

---

### 🤖 [Vision-Language-Action (VLA) Robot Arm Simulation](https://github.com/itsjoe3003/Vision-Language-Action-model)
`PyTorch` `OpenAI CLIP` `Panda-Gym` `Gymnasium`

Simplified VLA architecture built from scratch in PyTorch — separate vision embedder, language embedder, and action translation module using CLIP embeddings. Robot arm interprets natural language instructions (e.g., "move block left") and executes actions in a Panda-Gym simulation environment.

---

### 🖐️ [Hand Gesture Interactive Learning Platform](https://github.com/itsjoe3003/Hand-Gesture-interactive-Learning-Platform)
`MediaPipe` `OpenCV` `React` `TypeScript` `Flask` `MySQL`

Multimodal learning platform combining hand-gesture interaction (MediaPipe + custom keypoint classifier) with a full-stack web app. Features course delivery, MCQ/matching assignments, progress tracking, and a gesture-recognition pipeline.
